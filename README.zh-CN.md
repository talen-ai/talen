## 软件说明

Talen 是一款**轻量级、本地优先的知识库管理系统**，采用桌面客户端 + 本地 Web 服务的混合架构。安装后通过浏览器即可访问完整功能，所有数据——文档内容、向量嵌入、聊天记录——均使用 SQLite **存储在用户本地**。

与 Notion、Confluence 等在线知识管理工具不同，Talen 专注于**离线/本地部署场景**，数据完全由用户掌控，无需上传至第三方云端。与 Dify、FastGPT 等平台相比，Talen 更加轻量，安装更简单，几分钟内即可上手使用。

Talen 的核心基于 **RAG（检索增强生成）** 技术：文档被解析、切片并向量化后，用户提问时系统会从文档中检索最相关的片段，连同问题一起提交给 AI 大模型，生成基于实际文档内容的准确回答，有效避免 AI "幻觉"问题。

## 软件特性

### 多知识库管理
- 支持创建多个独立知识库，每个知识库拥有独立的文档、聊天和权限

### 多格式文档导入
- 支持 **PDF**、**Word（.docx）**、**Markdown（.md）**、**纯文本（.txt）**、**HTML**、**Excel（.xlsx）** 等常见格式
- 支持单文件导入和文件夹批量导入

### 自动化文档处理流水线
- 文档导入后自动完成文本提取、智能切片、关键词索引和向量索引构建

### AI 智能问答（RAG）
- 支持知识库内问答和跨知识库**全局智慧对话**
- 回答附带**来源标注**，直接链接到原文段落
- 流式输出，实时显示生成内容

### 全文检索
- **关键词搜索**：基于 SQLite FTS5 引擎，支持布尔操作符（AND、OR、NOT）
- **语义搜索**：基于向量相似度匹配
- **全局检索**：一次搜索跨越所有知识库

### FAQ 智能库
- 管理高频问答对，支持关键词匹配和分类筛选
- 支持 Excel/CSV 批量导入

### 多 AI 模型接入
- 支持 **OpenAI**、**百度文心**、**阿里通义**、**DeepSeek**、**Google Gemini** 等主流 AI 服务商
- 通过界面配置 API Key、Base URL、模型名称和参数

### 多用户与权限管理（专业版及旗舰版）
- 管理员与普通用户角色区分
- 按知识库设置读写/删除权限

### REST API 接口（企业版/旗舰版）
- 通过 HTTP API 将知识库能力开放给第三方系统集成

### 隐私与安全
- 所有数据本地存储，不向第三方云端上传
- 隐私密码保护聊天记录
- AES-256-GCM 加密授权文件

### 备份与恢复
- 完整数据备份（数据库、向量、配置）打包为单一 ZIP 文件
- 一键从备份文件完整恢复

## 技术架构

```
┌─────────────────────────────────────────────────────┐
│                  桌面客户端 (Tauri 2)                  │
│   系统托盘 · 右键菜单 · 开机自启 · 桌面通知 · 自动更新    │
├─────────────────────────────────────────────────────┤
│                本地 Web 服务 (Actix-web 4)              │
│   RESTful API · 文档处理流水线 · RAG 引擎               │
│   FTS5 搜索 · 向量存储 · 认证与授权                      │
├─────────────────────────────────────────────────────┤
│                   浏览器（UI 层）                        │
│   React 18 SPA · TailwindCSS · 响应式布局               │
├─────────────────────────────────────────────────────┤
│                    数据层 (SQLite)                      │
│   kb.db: 文档 · 切片 · 向量 · 聊天 · 用户               │
│   FTS5 全文索引 · BLOB 向量存储                          │
└─────────────────────────────────────────────────────┘
```

**文档处理流水线：**

1. **解析器（Parser）** — 根据文件类型选择解析器（PDF 用 lopdf、Word 用 docx-rs、Excel 用 calamine、HTML 用 html2text），提取纯文本
2. **切片器（Chunker）** — 将文本按段落、标题、字符数等策略切分为语义完整的文本块（chunks）
3. **向量嵌入（Embedder）** — 将每个文本块通过 AI 嵌入模型转换为向量
4. **索引器（Indexer）** — 为每个文本块建立 FTS5 全文搜索索引

**RAG 智能问答流程：**

```
用户提问
    ↓
[向量化 + 关键词检索]
    ↓
检索 Top-K 最相关的文档片段
    ↓
[文档片段 + 问题] → AI 大模型
    ↓
生成带有来源标注的回答
```

## 安装说明

Talen 以桌面应用形式提供，请从官方网站下载对应操作系统的安装包。

### 系统要求

- **操作系统**：Windows 10+、macOS 12+ 或 Linux（x86_64）
- **内存**：最低 4 GB（推荐 8 GB）
- **磁盘**：500 MB（应用本体）+ 文档存储空间
- **浏览器**：Chrome 90+、Firefox 90+、Edge 90+ 或 Safari 15+

### 安装步骤

1. 访问 [https://talen.ink](https://talen.ink) 下载对应平台的安装包
2. 运行安装程序，按照引导完成安装
3. 启动 Talen，系统将自动执行以下操作：
   - 自动初始化 SQLite 数据库
   - 首次运行引导设置管理员账号
   - 自动启动本地 Web 服务并打开默认浏览器
4. 开始创建知识库并导入文档
